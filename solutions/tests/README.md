# **Tests**

This folder contains unit tests for verifying the correctness of the solutions
provided in the `solutions/` folder.  
The tests are written using the **unittest** framework to cover a variety
of cases, including:

- ✅ Standard cases  
- ✅ Edge cases  
- ✅ Defensive cases
